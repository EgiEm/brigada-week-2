# 🧠 NLP Intent Classifier from Scratch

Weekly coursework, Jupyter Notebooks, and exercises completed as part of the **Brigada** technical training program. This repository documents the implementation of text classifiers from scratch (TF-IDF vectorizer + Logistic Regression) and their transition into robust, cross-validated production baselines.

## 🧠 What's Inside

- **TF-IDF from Scratch**: Customized text tokenization, Term Frequency (TF), Inverse Document Frequency (IDF), and TF-IDF calculation without library assistance.
- **Intent Router Pipeline**: Logistic Regression classifier implementation using `scikit-learn` to route queries into intents.
- **Cross-Validation & Error Analysis**: 5-fold cross-validation setup and confusion matrix analysis to verify model performance.
## 🛠️ Technologies Used

- **Languages**: Python, JavaScript (ES6)
- **Libraries**: Scikit-Learn, Pandas, NumPy
- **Tools**: Jupyter Notebook, Git
